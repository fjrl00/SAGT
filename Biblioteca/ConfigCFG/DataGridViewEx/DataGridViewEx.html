DatagridView personalizado

El ensamblado DataGridViewEx contiene una serie de propiedades y eventos que personalizan el manejo del control DataGridView de Windows Forms. Agrega propiedades que permiten determinar el tipo de datos que recibe la cuadrícula, la cantidad de caracteres que se pueden escribir en una celda y un menú contextual con las opciones básicas de edición, entre otros.

Además agrega dos eventos: EdiciónFinCopiar y KeyPressEditorCelda. El primero se ejecuta cuando se copia al portapapeles un grupo de celdas. El segundo evento se ejecuta cuando una celda se encuentra en edición y se presiona una tecla.

Descargar el ensamblado DataGridViewEx

Como cargar un DataGridViewEx en SharpDevelop

- Ejecute SharpDevelop e inicie una nueva solución del tipo Windows Forms.

- Seleccione la Caja de Herramientas y haga clic derecho la sección Componentes Personalizados.


- En el menú contextual seleccione el comando Personalizar Barra Lateral.

- En la ventana Configurar Barra Lateral seleccione la categoría Data y haga clic en el botón Añadir Componentes.

- En la ventana Añadir Componentes seleccione la pestaña Personalizado y haga clic sobre el botón que aparece al lado derecho de la celda Nombre de archivo. Seleccione el archivo DataGridViewEx.dll desde la carpeta donde lo haya colocado.

- Haga clic en el botón Mostrar Componentes para visualizar el ensamblado en la lista ubicada a la derecha.

Como usar el DataGridViewEx

En la barra de herramientas, en la sección Data, existe un componente llamado DatagridViewEx, listo para utilizarse en su proyecto Windows Form.

- Seleccione el componente DataGridViewEx y arrástrelo al formulario activo.

La cuadrícula que acaba de cargar es un DataGridView común y corriente, pero cuenta con algunas propiedades y eventos adicionales.

- Haga doble clic sobre el formulario y escriba el siguiente código en el evento MainFormLoad:

void MainFormLoad(object sender, EventArgs e)
{
             // Inicializar la cuadrícula
             dataGridViewEx1.NumeroColumnas = 4;
             dataGridViewEx1.NumeroFilas = 0;
             dataGridViewEx1.NumeracionFilas = true;
             // Configurar la columna 0
             dataGridViewEx1.Columns[0].HeaderText = "Solo texto";
             dataGridViewEx1.ColumnaAlineacion(0, DataGridViewContentAlignment.MiddleCenter);
             dataGridViewEx1.ColumnaMaximaLongitud(0, 2);
             dataGridViewEx1.ColumnaTipoEntrada(0, DataGridViewEx.TipoEntrada.SoloLetras);
             // Configurar la columna 1
             dataGridViewEx1.Columns[1].HeaderText = "Solo números";
             dataGridViewEx1.ColumnaAlineacion(1, DataGridViewContentAlignment.MiddleRight);
             dataGridViewEx1.ColumnaMaximaLongitud(1, 10);
             dataGridViewEx1.ColumnaTipoEntrada(1, DataGridViewEx.TipoEntrada.SoloNumeros);
             // Configurar la columna 2
             dataGridViewEx1.Columns[2].HeaderText = "Todos";
             dataGridViewEx1.ColumnaAlineacion(2, DataGridViewContentAlignment.MiddleRight);
             dataGridViewEx1.ColumnaMaximaLongitud(2, 10);
             dataGridViewEx1.ColumnaTipoEntrada(2, DataGridViewEx.TipoEntrada.SoloNumeros);
             // Configurar la columna 3
             dataGridViewEx1.Columns[3].HeaderText = "Bloqueada";
             dataGridViewEx1.ColumnaAlineacion(3, DataGridViewContentAlignment.MiddleRight);
             dataGridViewEx1.ColumnaMaximaLongitud(3, 10);
             dataGridViewEx1.Columns[3].ReadOnly = true; 
             
             // Establecer el número de filas
             dataGridViewEx1.NumeroFilas = 10;
}
